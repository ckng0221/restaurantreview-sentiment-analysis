About

The Restaurant Review Sentiment Analysis Tool is an online tool that can perform sentiment analysis specialized for restaurant reviews. The core of it is machine learning algorithms in classifying the input review text into multiclass ('positive', 'neutral', 'negative') and binary class ('positive', 'negative') sentiments. The tool also analyzes the linguistic features analytics of input text, including the POS tagging, lexicon polarity, and polarity rating.

The sentiment analysis tools used Logistic Regression machine learning model to perform sentiment analysis, where the model was the best identified model among Multinomial Naive Bayes, Support Vector Machine (SVM), and Random forest. The model was trained using Malaysia restaurant reviews data in different cities, and the textual reviews are processed using Natural Language Processing (NLP).

There are 2 ways the tool can be used:

  • Online : Input value on the textarea field on the web page to see the result.
  • REST API : Perform POST request on the API endpoint to get the result.

Using the Sentiment Analysis Tool, you will be getting both multiclass and binary sentiment analysis results:

Multiclass:
  • Positive
  • Neutral
  • Negative

Binary:
  • Positive
  • Negative

with the corresponding probabilities and for each sentiment, as well as the linguistic features.

The tool is developed for the Research Project of University Malaya as the fullfillment of the Master Degree of Data Science.
The tool and web app was developed by Ng Choon Khon (S2028941) in year 2022.
